# 🧠 Step 1 — Website Design

## 🎨 Tool

## কাজ কী?

Website UI design তৈরি করা।

Claude-কে শুধু idea explain করতে হয়।

---

## ✅ Easy Prompt

```text id="2nq7vx"
Design a modern SaaS landing page for my tool called "TaskFlow".

I want:
- Dark mode
- Blue glowing colors
- Clean modern layout
- Mobile-friendly design
- Strong signup buttons

Add:
- Hero section
- Features section
- Pricing
- Testimonials
- FAQ
- Footer

Style should look modern and startup-like.
```

---

## 🎯 Result

Claude:

* Full landing page structure বানাবে
* Modern UI দিবে
* Startup-style layout তৈরি করবে
