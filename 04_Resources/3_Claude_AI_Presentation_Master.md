# ১. Presentation Blueprint তৈরি করুন

Presentation শুরু করার আগে একটি clear structure দরকার।

এই prompt Claude-কে পুরো presentation map তৈরি করতে সাহায্য করবে।

## Improved Prompt

```text id="2j4vws"
Act as a professional presentation consultant.

Create a complete presentation blueprint for [Insert Topic].

Please define:
- Primary objective
- Target audience
- Core key message
- Recommended tone
- Slide-by-slide structure for a [Number]-slide presentation

Ensure:
- Logical flow
- Professional structure
- Audience engagement
- Clear storytelling progression
```

# ২. Story-Based Presentation তৈরি করুন

Boring slide এর বদলে storytelling-based presentation audience attention ধরে রাখে।

## Improved Prompt

```text id="4mn0pk"
Transform the topic [Insert Topic] into a compelling narrative using this structure:

Hook → Problem → Insight → Solution → Takeaway

Requirements:
- Emotionally engaging
- Data-driven
- Natural storytelling flow
- Strong transitions between sections
- Memorable ending

Make it feel like a journey instead of simple information sharing.
```

# ৩. Slide Architecture Design করুন

কোন slide এর পরে কোনটা আসবে এবং কেন আসবে, সেটা clear হওয়া খুব important।

## Improved Prompt

```text id="7ka3dt"
Design a detailed slide-by-slide architecture for a presentation on [Insert Topic].

For every slide include:
- Slide title
- Purpose of the slide
- Main talking point
- Suggested visual direction
- Transition into the next slide

Ensure the presentation maintains a seamless and professional flow.
```

# ১০. Executive Summary Slide তৈরি করুন

Presentation-এর শুরু বা শেষে একটি powerful summary slide থাকা জরুরি।

এটি পুরো বিষয়কে খুব কম সময়ে explain করে।

## Improved Prompt

```text id="0du7nm"
Synthesize the entire presentation into one powerful Executive Summary slide.

Include:
- Core problem
- Proposed solution
- Key insight
- Expected outcome
- Ultimate business or audience benefit

Requirements:
- Maximum clarity
- Minimum reading time
- Executive-level tone
- High-impact wording
```
