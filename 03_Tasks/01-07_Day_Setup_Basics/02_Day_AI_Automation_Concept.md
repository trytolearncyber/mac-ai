# Day 2: AI Automation Concept

## AI Automation কী?

AI Automation হলো এমন system যেখানে একটা trigger হলে AI নিজে থেকে multiple কাজ complete করে।

### Simple Flow:

Trigger → AI বুঝে → Decision নেয় → Action করে → Result

উদাহরণ:
নতুন email আসলো → AI category বুঝলো → Auto reply দিলো → Team কে notify করলো

---

# Agent কী?

Agent হলো এমন AI system যেটা goal পেলে নিজে plan করে, tools use করে, এবং কাজ complete করে।

* Prompt = একবার question → একবার answer
* Agent = Goal দিলে নিজে step-by-step কাজ করে

উদাহরণ:
“এই email টা লিখে দাও” = Prompt
“আমার inbox manage করো” = Agent

---

# Agent এর ৪টি Property

| Property       | সহজ ব্যাখ্যা                 | Real-Life Example                   |
| -------------- | ---------------------------- | ----------------------------------- |
| Autonomy       | নিজে কাজ করে                 | Robot vacuum নিজে cleaning করে      |
| Reactivity     | পরিস্থিতি বুঝে react করে     | Traffic signal দেখে গাড়ি থামে       |
| Proactivity    | আগে থেকেই action নেয়         | Battery low হলে charging এ যায়      |
| Social Ability | অন্যদের সাথে communicate করে | AI chatbot customer এর সাথে কথা বলে |

# Real-Life AI Automation Examples

| Trigger            | Automation Result                       |
| ------------------ | --------------------------------------- |
| Email আসে          | AI পড়ে → Reply পাঠায়                    |
| Form submit        | CRM save → Welcome email                |
| Order complete     | Invoice তৈরি → Email send               |
| Suspicious payment | Account freeze → SMS alert              |
| Customer message   | AI answer দেয় → Human support এ forward |

---

# Prompt vs Agent

| বিষয়        | Prompt           | Agent                  |
| ----------- | ---------------- | ---------------------- |
| কাজ         | একবার answer দেয় | পুরো task complete করে |
| Decision    | নেয় না           | নিজে decision নেয়      |
| Tools       | ব্যবহার করে না   | Tools ব্যবহার করে      |
| Human input | বারবার লাগে      | শুরুতে goal দিলেই হয়   |
